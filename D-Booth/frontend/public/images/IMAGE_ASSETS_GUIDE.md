## 🎨 图片处理建议

### 1. 统一风格
```
- 色温: 暖色调 (5000-6000K)
- 对比度: 中等偏高
- 饱和度: 自然鲜艳
- 锐度: 清晰但不过度
```

### 2. 优化规格
```bash
# 使用 ImageMagick 批量处理
convert input.jpg -resize 1920x1080^ -gravity center -extent 1920x1080 output.jpg

# 压缩优化
convert input.jpg -quality 85 -strip output.jpg
```

### 3. WebP 格式转换
```bash
# 转换为 WebP 以减小文件大小
cwebp -q 80 input.jpg -o output.webp
```

## 🚀 快速开始

### 方法1: 使用占位符服务（开发阶段）
```javascript
// 在组件中使用占位符
const placeholderImage = (width, height, text) => 
  `https://placehold.co/${width}x${height}/C4612F/FFFFFF?text=${encodeURIComponent(text)}`;

// 示例
<img src={placeholderImage(1920, 1080, 'Wedding Scene')} alt="Wedding" />
```

### 方法2: 下载真实图片
1. 访问 Unsplash/Pexels
2. 按照上述清单搜索并下载
3. 重命名文件按照规范
4. 放入对应目录

### 方法3: 生成 AI 图片
```bash

modern iPad setup with ring lights, warm lighting, joyful atmosphere,
```
